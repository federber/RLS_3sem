Формат входных данных:
	Данные по каждой трассе(массив параметров, параметры каждой КТ через пробел на новой строке) в отдельном .txt,
также есть .txt со списком всех доступных трасс(список файлов).

Данные по каждой серии измерений(массив КТ, параметры каждой КТ через пробел на новой строке) в отдельном .txt,
	также есть .txt со списком всех доступных серий измерений(список файлов).

Формат внутренних данных:
	std::map<size_t, std::vector<trace_point>> pattern_traces; - эталонные  трассы
	std::map<size_t, std::vector<inp_coord_point>> inp_meas_series; - серии измерений
	где ключ - номер трассы, значение - вектор координатный точек
	Векторы координатный точек отсортированы по времени (mod_time)
	координаты в необходимый момент времени можно искать бинарным поиском(для trace_point и inp_coord_point перегружен оператор <)
